Kebutuhan non-fungsional merupakan spesifikasi yang mendefinisikan atribut kualitas, batasan, serta karakteristik teknis sistem yang tidak berkaitan secara langsung dengan fungsi-fungsi spesifik, meliputi teknologi pengembangan, lingkungan perangkat lunak, dan spesifikasi perangkat keras yang diperlukan untuk mendukung operasional sistem secara keseluruhan.

a.	Teknologi Pengembangan Sistem
Sistem Informasi Penjualan UMKM ERNA Thrifting dibangun menggunakan framework Laravel 11 berbasis PHP untuk mengembangkan fungsi back-end. Laravel menggunakan pola arsitektur Model View Controller (MVC) yang menangani logika bisnis, routing URL, dan interaksi dengan basis data. Pada sisi front-end, implementasi tampilan dilakukan menggunakan Blade Template Engine milik Laravel bersama dengan teknologi pendukung HTML dan CSS. Framework CSS kustom digunakan untuk perancangan antarmuka yang responsif dengan dukungan Font Awesome 6 untuk ikonografi dan Google Fonts (Playfair Display serta Montserrat) untuk tipografi. jQuery digunakan untuk manipulasi DOM dan penanganan permintaan AJAX, sementara Chart.js dimanfaatkan untuk visualisasi grafik pada halaman dashboard. Untuk manajemen data, sistem menggunakan basis data relasional MySQL yang terintegrasi dengan Laravel melalui Eloquent Object Relational Mapping (ORM).

b.	Kebutuhan Perangkat Lunak
Perangkat lunak merupakan komponen fundamental yang mendukung operasional sistem secara menyeluruh. Kebutuhan perangkat lunak mencakup lingkungan pengembangan yang mendukung PHP versi 8.2 atau lebih tinggi, Laravel 11 sebagai framework utama, serta MySQL sebagai DBMS. Sistem juga memerlukan Composer sebagai dependency manager PHP dan Node.js beserta npm untuk manajemen aset front-end melalui Vite.
